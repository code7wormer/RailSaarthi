# Rail Saarthi 🚆

**The Smart Rail Assistant**

Smart navigation, real-time updates, and an effortless travel experience powered by Rail Saarthi. This project was developed as part of the Application Programming Project for the 2nd Semester.

## Features ✨
- **Live Tracking:** Precise GPS-based real-time train tracking system, accurate down to the minute.
- **Seat Matrix (Train Kundli):** Instantly check your seat confirmation and journey details with a single tap.
- **Smart Assistant (Trip Planner):** Navigate complex railway platforms and find amenities effortlessly.

## Tech Stack
- HTML5
- CSS3
- JavaScript
